# AIFFEL Quest Repository (Engineering Track)

이 레포지토리는 **AIFFEL Campus Online – Engineering 과정**에서 수행한  
퀘스트(Quest) 및 실습 프로젝트를 체계적으로 정리하기 위한 저장소입니다.

각 과정별 시간표를 기준으로 폴더를 구성하였으며,  
모든 퀘스트는 **CLI 환경에서 Git을 활용하여 관리**되었습니다.

---
